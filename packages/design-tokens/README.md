# @leta/design-tokens

Generated design tokens for the LETA design system.

## ⚠️ Do not hand-edit `dist/`

Files in `dist/` are generated by `@leta/cli` from the LETA Figma Library. To change a token, update it in Figma and run:

```bash
pnpm tokens:sync
```

Hand-edits will be flagged by the token-drift CI check and overwritten on the next sync.

## Source collections (Figma)

- **Brand** → primitives (raw colors, fonts, sizes)
- **Alias** → semantic tokens (themed: Neutral / Caution / Warning × moded: Product / Marketing)
- **Mapped Colors** → component-level color tokens
- **Mapped Type** → text styles
- **Mapped Sizes** → icon sizes

## Outputs

| File | Purpose |
|---|---|
| `dist/tokens.css` | CSS custom properties (with `[data-mode]` switching) |
| `dist/tokens.ts` | TypeScript constants |
| `dist/tailwind.preset.cjs` | Tailwind preset extending theme from CSS vars |
| `dist/text-styles.css` | Text style utility classes |
